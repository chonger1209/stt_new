# 屏幕时间追踪器更新说明 - 2025年11月25日

## 版本信息
- 版本号：v1.12.5
- 文件大小：12.5 MB
- 可执行文件：stt_new_20251125_single_line.exe

## 更新内容

### 日志格式优化
- 将"新增像素格记录"日志从三行格式改为一行格式
- 修改前：
  ```
  新增像素格记录: 
    notepad++.exe: 2个像素格, 总时长: 00:52 
   总计: 2个新增像素格 
  ```
- 修改后：
  ```
  新增像素格记录: notepad++.exe: 2个像素格, 总时长: 00:52; 总计: 2个新增像素格
  ```

### 技术细节
- 修改了 `log_new_pixels` 方法中的日志记录部分
- 使用分号分隔不同应用的信息
- 保留了所有原有的功能和数据记录
- 优化了日志可读性，减少了日志文件大小

## 安装说明
1. 下载 `stt_new_20251125_single_line.exe` 文件
2. 确保与 `config.txt` 和 `screen_time_tracker_new.ico` 文件在同一目录
3. 双击运行即可

## 注意事项
- 此版本优化了新增像素格记录的日志格式，使其更加紧凑
- 所有其他功能保持不变，包括新增像素格记录功能和数据持久化
- 如果有多个应用的新增像素格，它们将用分号分隔显示在同一行中
